# Open Source Export Manifest

This repository is a clean public export. It is not a history-preserving mirror
of any internal development repository.

## Included

- root package and TypeScript workspace configuration;
- AMCA source packages under `packages/`;
- public examples under `examples/`;
- public scenarios under `scenarios/`;
- public documentation under `docs/`;
- public CI and contribution files.

## Excluded

- internal evidence bundles;
- internal review reports;
- local provider run records;
- temporary branch artifacts;
- local machine paths;
- local service URLs;
- private phase-gate documents;
- local database URLs or credentials;
- generated coverage and test result artifacts.

## Public Maturity

AMCA Core is published as an alpha framework. Live integration and production
certification depend on explicit environment-specific setup and approval.
