# Governance

AMCA Core is maintained as a semantic governance framework for agentic systems.

Changes that affect protocol, proof, release, effects, ledger semantics, adapter
authority, security, or tenancy require tests and maintainer review.

Domain-specific behavior should live in extensions, examples, adapters, or
capabilities. It should not be added to AMCA Core unless a design decision shows
that the concept is generic and required by the governance model.
