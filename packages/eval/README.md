# @amca/eval

Regression and evaluation runner for AMCA.

## Maturity

This package produces deterministic regression and benchmark reports for AMCA
review workflows. Eval reports and benchmark output are not proof evidence and
must not be admitted as `EvidenceRef` support for released claims.

## Boundaries

- May compare replay and evaluation outcomes for regression detection.
- Must not mint evidence, receipts, proof verdicts, or release authority.
- Must not execute real adapters, tools, APIs, services, or domain workflows.
